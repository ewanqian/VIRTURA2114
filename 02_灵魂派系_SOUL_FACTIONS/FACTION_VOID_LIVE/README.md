## 🏴‍☠️ 派系档案

**“我们不是组织，我们是震荡之后还没消失的回声。”**

VOID-LIVE 是大崩溃后最早出现的派系之一。他们的前身是一群在2020-2030年代被迫转向地下的创意工作者、失业VJ和交互设计师。他们是被主流算法淘汰的“提案孤儿”。

在太空站，他们占据了**太空气球演出空间**的部分区域以及**Ghost Market**。

## 🧬 核心特征

*   **回声容器**：他们认为空间不应是结构，而是承载回声的容器。
*   **失败美学**：保留那些被标记为“错误”、“冗余”或“被否定”的数据。
*   **延迟响应**：他们的交互界面总是充满了延迟、残影和故障艺术（Glitch Art）风格。
