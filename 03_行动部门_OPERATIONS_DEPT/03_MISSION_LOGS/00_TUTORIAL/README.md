> **[HELPER_AI]**: 别紧张！这个任务是专门设计来展示我们系统能力的：
> - **自由探索**: 没有固定剧本，你想做什么都可以

> 最重要的是：**放松，享受过程**。

## 空间设定

**名称**: 大气层咖啡馆 (Atmosphere Café)
**时期**: 2025 年蓝图期
**状态**: 刚刚开业，充满理想主义
**氛围**: 温暖、包容、实验性

**核心理念**: 
"在数字空间中创造'第三空间'——不是家，不是工作，而是纯粹的社交和思考之地。"

---

## 主要区域

1. **前台区域**
   - 吧台、咖啡机、菜单板
   - 咖啡师 Luna 在这里工作

2. **座位区**
   - 舒适的沙发和桌椅
   - 几位顾客在聊天或独处
   - 墙上有艺术作品和留言板

3. **后院**
   - 安静的露台
   - 可以看到太空港的其他建筑
   - 有一个小型植物园（数字植物）

4. **地下室**（隐藏）
   - 需要特殊方式才能发现
   - 存放着咖啡馆的"秘密"

---

## 主要 NPC

### Luna（咖啡师）
- **性格**: 热情、理想主义、有点神秘
- **背景**: 咖啡馆的创始人之一，相信数字空间可以治愈孤独
- **特点**: 会记住客人的喜好，喜欢和人深入交谈
- **秘密**: 她曾经是"虚拟航线"项目的参与者

### 其他顾客（3-5 人）
- 随机生成，但要有个性
- 可能包括：艺术家、程序员、哲学家、迷失者等
- 他们之间可能有互动

---

## 互动规则

1. **自由度**
   - 质检员可以做任何事：点咖啡、聊天、探索、甚至尝试破坏
   - 不要限制他们的行动，但要给出合理的反馈

2. **对话系统**
   - NPC 应该能理解复杂的对话
   - 记住之前的对话内容
   - 根据质检员的态度调整反应（友好/警惕/好奇）

3. **环境响应**
   - 质检员的行动会影响环境（如：打翻咖啡、移动家具）
   - 其他 NPC 会注意到并反应

4. **秘密与彩蛋**
   - 如果质检员表现出好奇心，给他们线索
   - 地下室的入口需要通过对话或观察发现
   - 彩蛋：Luna 的日记，提到"虚拟航线"

5. **意外事件**
   - 随机触发一些小事件（如：停电、新客人到来、咖啡机故障）
   - 这些事件应该增加趣味性，而不是阻碍

## 🎮 GAMEPLAY_TIPS // 游玩提示

### 如何开始
1. 复制上面的 Prompt
2. 粘贴到 AI 对话框
3. 等待开场描述
4. 开始输入你的行动！

### 输入示例

**简单输入**:
- "我走向吧台"
- "和咖啡师聊天"
- "点一杯咖啡"

**复杂输入**:
- "我走向吧台，观察咖啡师的表情，试图判断她的心情"
- "我假装不经意地问：'这里以前是做什么的？'"
- "我环顾四周，寻找任何不寻常的细节"

**创意输入**:
- "我尝试黑进咖啡机的系统"
- "我开始在留言板上画画"
- "我假装是卫生检查员"

### 展示系统优势的方法

#### 1. 测试对话深度
尝试和 Luna 进行哲学讨论：
- "你觉得数字空间中的孤独和现实中的孤独有什么不同？"
- "如果这个空间有一天消失了，它存在过吗？"

#### 2. 测试记忆系统
在对话中提到一些细节，稍后再提起，看 AI 是否记得：
- "我叫 [你的名字]，我是来做质检的"
- 稍后："还记得我之前说的吗？"

#### 3. 测试环境响应
做一些"出格"的事，看系统如何反应：
- "我故意打翻咖啡"
- "我大声唱歌"
- "我尝试进入员工区域"

#### 4. 测试意外处理
提出 AI 可能没有预设的要求：
- "我想举办一个即兴诗歌朗诵会"
- "我要求 Luna 教我做咖啡"
- "我邀请所有顾客玩一个游戏"

## 📊 EVALUATION_CRITERIA // 评估标准

任务结束后，你需要评估这个空间。考虑以下方面：

### 1. 沉浸感 (1-5 分)
- 描述是否生动？
- 你是否感觉"在场"？

### 2. 互动质量 (1-5 分)
- NPC 的对话是否自然？
- 系统是否理解你的意图？

### 3. 自由度 (1-5 分)
- 你能做你想做的事吗？
- 系统是否限制了你的行动？

### 4. 意外性 (1-5 分)
- 是否有惊喜？
- 是否有不可预测的事件？

### 5. 情感共鸣 (1-5 分)
- 你是否对角色或空间产生了感情？
- 是否有令你印象深刻的时刻？
