# NPC联系方式 // 派系关键人物档案

> **[PURPOSE]**: 在AI游戏中可以"联系"这些NPC  
> **[USAGE]**: 复制联系信息到AI对话中，发起互动

---

## 使用方法

在AI游戏中输入：
```
我想联系 [NPC名字]
发送消息：[你的消息]
```

或者：
```
我前往 [NPC位置]，寻找 [NPC名字]
```

## 如何触发深度对话

1. **建立信任**: 多次互动，完成他们的请求
2. **展示诚意**: 分享你自己的故事
3. **问对问题**: 不要问太直接的问题，从侧面入手
4. **尊重边界**: 如果他们不想回答，不要强迫
