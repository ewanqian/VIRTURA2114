## 中央诊所

**位置**: 疗愈松林核心区  
**功能**: 主要治疗设施  
**开放时间**: 全天候

### 区域划分

#### 1. 接待大厅

```
    ┌─────────────────────────────────────────────────────────┐
    │                                                         │
    │   接 待 大 厅                                          │
    │   ───────────                                          │
    │                                                         │
    │   功能：                                                │
    │   - 初次来访登记                                       │
    │   - 紧急分诊                                           │
    │   - 等候区                                             │
    │                                                         │
    │   设计理念：                                            │
    │   温暖、安全、不像"医院"                               │
    │   柔和的光线，舒适的座椅，                             │
    │   墙上展示着康复者的艺术作品                           │
    │                                                         │
    └─────────────────────────────────────────────────────────┘
```

**访客体验**：

> "我第一次来的时候很紧张。
> 但接待大厅不像我想象的那种冷冰冰的诊所。
> 有柔和的音乐，有植物，有一只虚拟的猫在角落里睡觉。
> 我开始放松了。"

#### 2. 评估室

**功能**: 初步评估和诊断  
**设备**: 意识扫描仪、碎片检测器、ARIA-7终端

**评估流程**：

```
    第1步：基础信息采集（15分钟）
    ↓
    第2步：意识状态扫描（30分钟）
    ↓
    第3步：碎片检测（如适用，45分钟）
    ↓
    第4步：与治疗师初谈（60分钟）
    ↓
    第5步：制定初步治疗计划
```

#### 3. 治疗室

**数量**: 12间  
**类型**: 
- 标准治疗室（8间）
- 碎片对话室（2间）：配备多人格同时呈现技术
- 创伤处理室（2间）：隔音，可调节环境

**治疗室设计**：

> "每间治疗室都可以根据患者需求调整。
> 灯光、声音、温度、甚至虚拟环境都可以定制。
> 有的患者需要明亮开放的空间，
> 有的需要安全封闭的角落。
> 我们尊重每个人的需求。"

#### 4. 碎片整合中心

**功能**: 进行复杂的碎片整合治疗  
**设备**: 高级意识同步设备、碎片容器阵列

**特殊说明**：

```
╔═══════════════════════════════════════════════════════════════╗
║                                                               ║
║   碎片整合中心使用须知                                        ║
║   ─────────────────────                                      ║
║                                                               ║
║   1. 整合治疗需要提前预约                                     ║
║   2. 治疗前24小时需要禁食（数字和物理）                       ║
║   3. 治疗后需要在观察室休息至少4小时                          ║
║   4. 整合不是强制的——只有在患者明确要求时才进行              ║
║                                                               ║
╚═══════════════════════════════════════════════════════════════╝
```

#### 5. 观察室

**功能**: 治疗后休息和监测  
**容量**: 20个床位

**环境**：
- 柔和的自然光模拟
- 可调节的隔音屏障
- 每个床位都有独立的环境控制

#### 6. 研究实验室

**功能**: 进行意识研究和新治疗方法开发  
**负责人**: Dr. Marcus Chen

**当前研究项目**：

| 项目 | 状态 | 目标 |
|------|------|------|
| 碎片预防 | 进行中 | 开发防止意识碎片化的方法 |
| 远程整合 | 测试中 | 允许在物理距离外进行碎片整合 |
| 创伤疫苗 | 早期 | 增强意识对创伤的抵抗力 |

### 急救服务内容

**可处理的紧急情况**：
- 急性意识碎片化
- 严重数字创伤
- 身份崩溃
- 情感过载
- 记忆急性丢失

**急救流程**：

```
    1. 呼叫急救（语音/数字信号）
       ↓
    2. 急救员5分钟内到达
       ↓
    3. 现场稳定（防止进一步损伤）
       ↓
    4. 评估是否需要转移
       ↓
    5a. 轻度：现场处理后观察
    5b. 重度：转移至中央诊所
```

## 访客须知

### 第一次来访

如果你是第一次来到模组临床：

1. **不要紧张** - 我们不是传统意义上的"医院"
2. **带上你的故事** - 我们想听你的经历
3. **没有"正确答案"** - 我们不会评判你
4. **可以随时离开** - 你有完全的自主权

### 常见问题

**Q: 我需要预约才能来吗？**
A: 建议预约，但我们也接受walk-in。紧急情况无需预约。

**Q: 治疗会痛吗？**
A: 物理上不会。情感上，面对自己的碎片可能会不舒服，但我们会全程陪伴你。

**Q: 我的隐私会被保护吗？**
A: 绝对会。我们有严格的隐私政策，你的信息不会被分享给任何第三方。

**Q: 如果我中途想停止治疗呢？**
A: 你随时可以停止。我们尊重你的决定。
