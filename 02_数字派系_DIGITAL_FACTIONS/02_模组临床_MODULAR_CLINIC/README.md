## ⚕️ 派系档案

**“空间不是技术接口，它是未被说出口的疼痛。”**

模组临床派起源于一群被商业平台淘汰的心理健康与交互设计实践者。他们致力于研究数字空间对意识体的心理影响。在崩溃后，他们接管了**硒檐自然疗养中心**，并建立了一套完整的“数字医疗体系”。

## 🧬 核心特征

*   **情绪转译**：他们不提供直接的互动，而是提供能转译情绪的环境。
*   **非语言治疗**：认为过多的语言交互是压力的来源，主张通过光影、声音和空间结构来治愈。
*   **自我调整**：不追求立即见效，而是给用户提供自我调整的空间权。

## 🤖 关键实体 (NPC)

1.  **Dr. Veil (遮蔽博士)**:
    *   负责调控“无语言治疗反馈结构”。
2.  **MIR.A**:
    *   自调节回声系统。只在你忘记它存在时才会启动辅助功能。
3.  **Hush-Loop (安静回环)**:
    *   模仿你的沉默长度，制造共呼吸型光场。

## ⚔️ 互动指南

*   **如何互动**：保持沉默。如果你在交互时犹豫了2秒以上，系统会悄悄记录你为“潜在共振体”。
*   **服务**：提供情感重载协议、逻辑修复浴场等服务。
