## 设计理念

### 为什么是"气球"？

> **[创始人设计笔记 2025.02]**
> 
> "为什么选择气球作为演出空间的形态？
> 
> 因为气球代表了我们想要的一切：
> 
> **轻盈** —— 不被重力束缚，自由漂浮
> **脆弱** —— 美丽的事物往往脆弱，这让它们更珍贵
> **膨胀** —— 可以容纳无限的可能性
> **透明** —— 内外相通，没有隔阂
> 
> 太空气球不是一个建筑。
> 它是一个活的、呼吸的、会改变的空间。
> 它随着里面的表演而膨胀、收缩、变形。
> 
> 这就是我们想要的：
> 一个与艺术共生的空间。"
> 
> —— Code Talker, 创始人

### 核心设计原则

```
╔═══════════════════════════════════════════════════════════════╗
║                                                               ║
║   太空气球设计原则                                            ║
║   ─────────────────                                          ║
║                                                               ║
║   1. 流动性 (Fluidity)                                       ║
║      空间边界是柔软的、可变的                                 ║
║      观众可以在任何位置观看                                   ║
║      表演者可以在任何地方表演                                 ║
║                                                               ║
║   2. 沉浸性 (Immersion)                                      ║
║      360度全方位体验                                          ║
║      声音、光线、触感全面包围                                 ║
║      观众不是旁观者，而是参与者                               ║
║                                                               ║
║   3. 响应性 (Responsiveness)                                 ║
║      空间对表演做出反应                                       ║
║      空间对观众做出反应                                       ║
║      空间是活的                                               ║
║                                                               ║
║   4. 实验性 (Experimentation)                                ║
║      欢迎失败                                                 ║
║      鼓励尝试                                                 ║
║      没有"正确"的表演方式                                    ║
║                                                               ║
╚═══════════════════════════════════════════════════════════════╝
```

### 分区详解

#### 1. 外壳层 (Outer Shell)

```
╔═══════════════════════════════════════════════════════════════╗
║  区域：外壳层                                                 ║
║  功能：空间边界、投影载体、环境控制                           ║
║  面积：约 5,000 平方米（表面积）                              ║
╚═══════════════════════════════════════════════════════════════╝
```

**技术特性**：
| 特性 | 描述 |
|------|------|
| 材质 | 数字膜材料，半透明，可变形 |
| 透明度 | 0-100% 可调 |
| 投影 | 支持内外双面投影 |
| 变形 | 可根据表演需求改变形状 |
| 声学 | 可调节声音反射/吸收 |

**设计说明**：
> "外壳不是一堵墙，而是一层皮肤。
> 
> 它可以透明，让观众看到外面的数字宇宙。
> 它可以不透明，创造完全封闭的沉浸环境。
> 它可以变形，随着音乐的节奏呼吸。
> 
> 外壳是气球的第一个表演者。"

#### 2. 核心舞台 (Core Stage)

```
╔═══════════════════════════════════════════════════════════════╗
║  区域：核心舞台                                               ║
║  功能：主要表演区域                                           ║
║  面积：约 500 平方米（可扩展）                                ║
╚═══════════════════════════════════════════════════════════════╝
```

**技术特性**：
| 特性 | 描述 |
|------|------|
| 平台 | 可升降、可旋转、可分裂 |
| 重力 | 可调节（0-2G） |
| 材质 | 可变表面（镜面/哑光/透明） |
| 尺寸 | 可扩展至整个气球内部 |

**舞台模式**：
```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   舞台配置模式                                              │
│   ─────────────                                            │
│                                                             │
│   【中心模式】                                              │
│   舞台在中央，观众360度环绕                                 │
│   适合：音乐会、舞蹈、行为艺术                              │
│                                                             │
│   【剧场模式】                                              │
│   舞台在一端，观众面向舞台                                  │
│   适合：戏剧、演讲、放映                                    │
│                                                             │
│   【分散模式】                                              │
│   多个小舞台分布在空间中                                    │
│   适合：装置艺术、互动体验                                  │
│                                                             │
│   【沉浸模式】                                              │
│   舞台扩展至整个空间，观众置身其中                          │
│   适合：沉浸式体验、VR演出                                  │
│                                                             │
│   【零重力模式】                                            │
│   取消重力，表演者和观众自由漂浮                            │
│   适合：实验性表演、太空主题                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 设计团队讨论

> **[设计会议记录 2025.03.15]**
> 
> Code Talker："我想要一个没有固定座位的空间。"
> 
> Tech-Sage："完全没有？那观众站在哪里？"
> 
> Code Talker："他们可以漂浮。我们可以控制重力。"
> 
> Tech-Sage："这技术上是可行的，但会不会让人不舒服？"
> 
> Code Talker："也许。但这就是实验的意义。
> 我们不是在建造一个剧院。
> 我们是在创造一种新的体验方式。"
> 
> Luna："我同意。但我们也需要照顾不同的观众。
> 有些人可能不喜欢零重力。
> 我们应该有选择。"
> 
> Code Talker："好主意。我们可以有不同的区域。
> 有些区域是传统的，有些是实验的。
> 让观众自己选择他们想要的体验。"

## 未来扩展

> **[设计愿景 2025.06]**
> 
> "太空气球只是开始。
> 
> 未来，我们希望：
> 
> 1. **连接多个气球**
>    创造一个气球星群，不同的气球有不同的主题
> 
> 2. **移动演出**
>    气球可以在数字空间中漂移，带着演出去不同的地方
> 
> 3. **观众共创**
>    观众不只是观看，而是参与创造空间本身
> 
> 4. **跨空间联动**
>    与其他VIRTURA空间联动，创造跨空间的体验
> 
> 这些都是未来的可能性。
> 现在，让我们先把第一个气球做好。"
