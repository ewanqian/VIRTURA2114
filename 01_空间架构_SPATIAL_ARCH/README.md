## 概述 Overview

VIRTURA 太空站并非单一维度的存在。它由一系列精心设计的**“功能扇区”**组成，每个扇区都拥有独特的视觉语言与情感氛围。

我们曾在 2022-2023 年间基于 Unity 引擎构建了 VRChat 版本的早期地图。随着 2050 年大崩溃设定的引入，这些旧地图现已作为**“历史遗迹”**封存，我们正转向利用 AI 技术生成更具沉浸感、更动态的新一代空间。

## 📐 设计哲学 Design Philosophy

我们相信空间不应只是互动的背景，而是**叙事的主体**。

1.  **情感容器 (Emotional Vessel)**: 建筑应能承载并放大使用者的情绪。
2.  **虚实共生 (Phygital Symbiosis)**: 探索物理规则与数字逻辑的边界。
3.  **动态演化 (Dynamic Evolution)**: 空间不是静止的，它会随着派系控制和玩家行为而改变面貌。
