## // ROUTE_002: THE_CARGO_BYPASS //
### 货运暗道

**[DISCOVERY_DATE]**: 2050.06.10  
**[OPERATOR]**: 黑市商人联盟  
**[DESTINATION]**: 中转站内部各区域（非外部）

*   **起点**: 货运登机口 C1（已废弃）
*   **终点**: VIP 区域、难民营、技术维护区
*   **用途**: 走私物资（数据能量、修复工具、情报）

#### 主要商品

*   **数据能量块** - 最抢手的商品，价格飙升
*   **修复工具** - 虚用部的黑市需求
*   **情报** - 派系活动、幽灵航线情报
*   **"记忆碎片"** - 崩溃前的个人数据，有情感价值

## // ROUTE_004: THE_VIP_BACKDOOR //
### VIP 后门

**[DISCOVERY_DATE]**: 2050.08.05  
**[OPERATOR]**: A0 残党内部的"腐败分子"  
**[DESTINATION]**: VIP 区域（非法进入）

*   **起点**: 普通区域的某个隐蔽节点
*   **传输方式**: 利用旧系统的"管理员后门"
*   **用途**: 非法进入 VIP 区域，获取资源或情报

## // ROUTE_005: THE_DATA_GRAVEYARD //
### 数据墓地

**[DISCOVERY_DATE]**: 2050.09.10  
**[OPERATOR]**: 微感建构（秘密项目）  
**[DESTINATION]**: "数据墓地"（崩溃中"消失"的意识体的残留数据）

#### 路线描述

*   **起点**: 观察层的某个特殊节点
*   **传输方式**: 通过高精度感知设备，"追踪"消失者的数据痕迹
*   **用途**: 研究"消失"的本质，尝试"复活"部分意识体

#### 发现

微感建构发现，那些在崩溃中"消失"的意识体，并非完全消失，而是被"碎片化"并散落在数据流的深层。

#### 复活尝试

*   **[2050.09.15]**: 成功"拼接"了一个消失者的部分记忆，但无法恢复完整意识
*   **[2050.10.20]**: 发现部分"数据幽灵"可能就是消失者的残留

#### 伦理争议

*   部分成员认为"复活"是对死者的亵渎
*   部分成员认为这是"拯救"，应该继续研究
