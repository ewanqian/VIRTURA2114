## 分区详解

### 1. 感知花园 (Sensory Garden)

```
╔═══════════════════════════════════════════════════════════════╗
║  区域名称：感知花园                                           ║
║  英文名称：Sensory Garden                                     ║
║  面积：约 500 平方米（数字空间）                              ║
║  容量：无限制                                                 ║
║  预约：无需预约                                               ║
╚═══════════════════════════════════════════════════════════════╝
```

**功能定位**：入口过渡区，帮助访客从外部世界的喧嚣中平静下来

**空间特点**：

| 感官 | 设计元素 | 作用 |
|------|----------|------|
| 视觉 | 柔和的绿色调、渐变光线 | 降低视觉刺激 |
| 听觉 | 轻柔的风声、远处的鸟鸣 | 替换城市噪音 |
| 触觉 | 温暖的地面、微风触感 | 身体放松 |
| 嗅觉 | 淡淡的草木香 | 唤起自然记忆 |

**空间流程**：

```
    进入
      │
      ▼
    ┌─────────────────────────────────────┐
    │  1. 光线过渡区                       │
    │     从明亮逐渐变为柔和               │
    │     时长：约2分钟                    │
    └─────────────────────────────────────┘
      │
      ▼
    ┌─────────────────────────────────────┐
    │  2. 情绪检测点                       │
    │     系统会轻柔地询问当前状态         │
    │     "你今天感觉如何？"               │
    └─────────────────────────────────────┘
      │
      ▼
    ┌─────────────────────────────────────┐
    │  3. 花园漫步                         │
    │     自由探索各种植物和小径           │
    │     建议时长：10-15分钟              │
    └─────────────────────────────────────┘
      │
      ▼
    前往下一区域
```

**设计理念**：

> "感知花园不是目的地，而是一扇门。
> 
> 它帮助访客'卸下'外部世界的压力，
> 准备好进入更深层的疗愈空间。
> 
> 就像进入寺庙前要净手，
> 进入疗愈空间前要'净心'。"
> 
> —— Dr. Elena Moss, 设计笔记 (2025.04)

### 2. 松林漫步道 (Forest Walk Path)

```
╔═══════════════════════════════════════════════════════════════╗
║  区域名称：松林漫步道                                         ║
║  英文名称：Forest Walk Path                                   ║
║  面积：约 2000 平方米（数字空间）                             ║
║  容量：同时最多50人                                           ║
║  预约：无需预约                                               ║
╚═══════════════════════════════════════════════════════════════╝
```

**功能定位**：主要疗愈空间，提供沉浸式自然体验

**空间特点**：

- **松树林**：高大的松树，树冠遮蔽天空，光线斑驳
- **小径**：蜿蜒的土路，有落叶和松针
- **声音**：风吹松涛、鸟鸣、远处溪流声
- **冥想节点**：沿途设有7个冥想点，每个有不同主题

**冥想节点详情**：

| 节点 | 主题 | 引导内容 |
|------|------|----------|
| 1 | 呼吸 | 专注于呼吸的进出 |
| 2 | 身体 | 感知身体各部位的状态 |
| 3 | 声音 | 聆听周围的自然声音 |
| 4 | 思绪 | 观察思绪如云朵飘过 |
| 5 | 情感 | 允许情感自然流动 |
| 6 | 感恩 | 回想值得感恩的事物 |
| 7 | 意图 | 设定一个温柔的意图 |

**特殊机制**：

```
    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   "迷路"机制                                               │
    │   ─────────                                                │
    │                                                             │
    │   松林漫步道没有固定的路线。                                │
    │   访客可能会"迷路"——但这是设计的一部分。                  │
    │                                                             │
    │   "迷路"时，系统不会立即指引方向。                         │
    │   而是鼓励访客：                                            │
    │   - 停下来                                                  │
    │   - 观察周围                                                │
    │   - 感受当下                                                │
    │                                                             │
    │   当访客真正放松时，出口会自然出现。                        │
    │                                                             │
    │   这个机制教导人们：                                        │
    │   有时候，"不知道去哪里"是可以的。                         │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘
```

### 3. 核心治疗区 (Core Therapy Zone)

#### 3.1 静默疗愈室 (Silence Chamber)

```
╔═══════════════════════════════════════════════════════════════╗
║  区域名称：静默疗愈室                                         ║
║  容量：同时最多10人                                           ║
║  预约：建议预约                                               ║
║  时长：建议30-60分钟                                          ║
╚═══════════════════════════════════════════════════════════════╝
```

**功能定位**：完全安静的空间，适合深度冥想和自我反思

**空间特点**：
- 绝对静音（屏蔽所有外部声音）
- 柔和的暖色光线
- 舒适的坐垫和靠背
- 没有任何视觉干扰

**使用指南**：

> "静默疗愈室是给那些需要'什么都不做'的人准备的。
> 
> 在这里，你不需要冥想、不需要思考、不需要'做'任何事。
> 你只需要存在。
> 
> 静默本身就是疗愈。"

#### 3.2 声音疗愈室 (Sound Therapy Room)

```
╔═══════════════════════════════════════════════════════════════╗
║  区域名称：声音疗愈室                                         ║
║  容量：同时最多20人                                           ║
║  预约：需要预约                                               ║
║  时长：固定疗程45分钟                                         ║
╚═══════════════════════════════════════════════════════════════╝
```

**功能定位**：使用特定频率的声音进行疗愈

**疗程类型**：

| 疗程 | 频率 | 作用 | 适合人群 |
|------|------|------|----------|
| 放松 | 432Hz | 降低焦虑 | 压力大的人 |
| 专注 | 528Hz | 提升专注力 | 注意力分散的人 |
| 睡眠 | 174Hz | 促进深度睡眠 | 失眠者 |
| 情绪 | 639Hz | 平衡情绪 | 情绪波动的人 |

#### 3.3 深度治疗室 (Deep Therapy Room)

```
╔═══════════════════════════════════════════════════════════════╗
║  区域名称：深度治疗室                                         ║
║  容量：一对一                                                 ║
║  预约：必须预约                                               ║
║  时长：60-90分钟                                              ║
╚═══════════════════════════════════════════════════════════════╝
```

**功能定位**：处理深层心理问题，需要专业治疗师陪同

**服务内容**：
- 创伤处理
- 情绪释放
- 认知重构
- 关系修复

**预约流程**：

```
    1. 提交申请
       ↓
    2. 初步评估（在线问卷）
       ↓
    3. 匹配治疗师
       ↓
    4. 确认时间
       ↓
    5. 进行治疗
       ↓
    6. 后续跟进
```

### 4. 水疗区 (Water Therapy Zone)

#### 4.1 溪流漫步 (Stream Walk)

**功能定位**：在溪流旁行走，感受水的流动

**特点**：
- 可以"踩"进溪水（感知模拟）
- 水温可调节
- 有小鱼游过

#### 4.2 瀑布冥想 (Waterfall Meditation)

**功能定位**：在瀑布声中进行冥想

**特点**：
- 瀑布的白噪音有助于清空思绪
- 可以"站"在瀑布下（感知模拟）
- 水雾的清凉感

#### 4.3 温泉浸泡 (Hot Spring)

**功能定位**：深度身体放松

**特点**：
- 温暖的水温（感知模拟）
- 矿物质的香气
- 可以选择独处或与他人共享
