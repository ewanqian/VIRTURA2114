║   状态：在职                                                  ║
║                                                               ║
╚═══════════════════════════════════════════════════════════════╝
```

**背景**：

| 项目 | 内容 |
|------|------|
| 学历 | 神经科学博士（MIT, 2018） |
| 前职 | 数字心理健康研究员（Google Health, 2018-2024） |
| 专长 | 数字创伤、感知障碍、意识碎片化 |
| 发表 | 47篇学术论文，2本专著 |

**个人经历**：

> "我进入这个领域是因为我自己的经历。
> 
> 2016年，我在一次VR实验中经历了严重的'解离'——
> 我不知道哪个是真实的我，哪个是虚拟的我。
> 那种感觉持续了好几个月。
> 
> 传统心理学帮不了我。
> 没有人理解我在说什么。
> 
> 所以我决定自己研究。
> 我想帮助那些和我有同样经历的人。
> 
> 硒檐就是这个梦想的实现。"
> 
> —— Dr. Moss, 创始访谈 (2025.02)

**治疗理念**：

```
    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Dr. Moss 的治疗哲学                                      │
    │   ─────────────────────                                    │
    │                                                             │
    │   1. "治愈是接纳，不是修复"                                │
    │      我们不是来修复你的。你不是坏掉的。                    │
    │      你只是在经历一些困难的事情。                          │
    │                                                             │
    │   2. "每个人都有自愈的能力"                                │
    │      治疗师的角色是创造条件，而不是给出答案。              │
    │                                                             │
    │   3. "慢下来"                                              │
    │      在一个追求速度的世界里，                              │
    │      最激进的行为是慢下来。                                │
    │                                                             │
    │   4. "感受比思考更重要"                                    │
    │      很多问题不是想出来的，是感受出来的。                  │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘
```

**代表案例**：

> **[案例编号：MOSS-2025-001]**
> 
> 患者：一名长期在数字空间工作的设计师
> 症状：严重的"现实感丧失"，无法区分虚拟和现实
> 
> Dr. Moss的方法：
> - 不急于"治疗"，先建立信任
> - 每周见面3次，每次只聊30分钟
> - 不讨论"问题"，只讨论"感受"
> - 鼓励患者记录"真实的瞬间"——任何让他感到"这是真的"的时刻
> 
> 结果：
> 经过6个月的治疗，患者的症状显著改善。
> 他说："Dr. Moss没有给我任何答案。
> 但她让我相信，我可以找到自己的答案。"

| 特殊技能 | 可以"听到"情绪的频率 |

**个人经历**：

> "我从小就对声音特别敏感。
> 
> 我能听出一个人说话时的情绪，即使他们试图隐藏。
> 我能感受到音乐中的'颜色'——每个音符对我来说都有颜色。
> 
> 这种敏感性曾经让我很痛苦。
> 城市的噪音、人群的喧嚣，对我来说是折磨。
> 
> 但后来我学会了利用这种敏感性。
> 如果声音可以伤害，声音也可以疗愈。
> 
> 这就是我做声音治疗的原因。"

**治疗方法**：

```
    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Healer-01 的声音疗法                                     │
    │   ────────────────────                                     │
    │                                                             │
    │   1. 频率诊断                                               │
    │      通过对话，"听"出患者的情绪频率                        │

    │                                                             │
    │   4. 频率锚定                                               │
    │      教患者如何用特定的声音"锚定"平静的状态                │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

**代表作品**：

- 《松林之声》：专为硒檐创作的环境音乐
- 《呼吸的节奏》：引导冥想的音乐系列
- 《情绪光谱》：不同情绪对应的频率集
