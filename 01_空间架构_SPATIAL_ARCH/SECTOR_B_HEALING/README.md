## 🏛️ 空间概览

在充满数据噪点和高频信息的太空港中，硒檐（Seleiumeaves）是一片难得的净土。这里由**模组临床派（Modular Clinic Union）**精心维护，旨在为遭受“数字过载”或“逻辑创伤”的意识体提供修复服务。

整个空间被设计为一片**数字化松林景观**，充满了东方禅意与未来医疗技术的结合。

## ⚙️ 核心特征

*   **情绪疫苗接种**：
    *   针对“信息焦虑”、“算法偏见”等数字疾病的预防性数据包。

*   **生物反馈环境**：
    *   空间内的光照、气流（数据流速）、声音会根据访客的实时生理/心理指标自动调节。
    *   如果你感到焦虑，周围的松林会变得更加茂密以提供安全感。

*   **Regen Grove (再生林地)**：
    *   这是崩溃后新增的深层区域。树影会随着你脑中词汇的频率变化。
    *   是模组临床派的核心实验场，用于修复严重受损的代码人格。

## 📍 关键区域

1.  **星语轩 (Whisper Pavilion)**：用于深度冥想的悬浮庭院。
2.  **浮空温泉池 (Data Springs)**：充满了高密度修复数据流的池塘，浸泡其中可恢复逻辑完整性。
3.  **咨询室 (The Clinic)**：Dr. Veil 进行无语言治疗的场所。

## 👥 相关派系：模组临床派

他们不提供互动，只提供“情绪转译环境”。在这里，你的沉默和犹豫会被视为一种需要被呵护的症状，而不是系统错误。
