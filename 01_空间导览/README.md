## 📝 关于旧城

> 旧城曾是VIRTURA太空站的绝对中心。2049年前，太空气球是整个数字空间最著名的演出场地，商业和文化活动都围绕它展开。
>
> 2050年大崩溃后，这里遭受了最严重的破坏。如今，旧城已被划为历史保护区，部分区域仍在修复中。但它承载的记忆，让每一位访客都会来此朝圣。
>
> 新的城市中心已转移到中央区，但旧城的某些商铺和展馆仍在运营，展示着黄金时代的辉煌。
