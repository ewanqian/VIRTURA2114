## 区域一览

### [01 港区](01_港区/) — 入口与交通
你到达VIRTURA的第一站。

| 地点 | 内容 |
|------|------|
| **接待中心** | 新手引导、杂志架、咨询 |
| **港务区** | 航站楼、航空公司、航班 |
| **海关** | 入境检查、申报 |
| **货运码头** | 收发货、仓储 |
| **工业区** | AI工厂、数据中心（不开放） |

---

## 建议路线

### 半日游（3小时）
接待中心 → 中央区逛街 → 文化街区咖啡

### 一日游（6小时）
港区 → 中央区（午餐）→ 绿带公园 → 旧城遗址

### 深度游（2-3天）
Day 1: 港区 + 中央区
Day 2: 绿带（博物馆、疗愈）
Day 3: 旧城 + 自由活动
