## 简介

VIRTURA水族馆收藏了数字空间里的海洋生物。

有些是模拟真实物种，有些是这里独有的——只存在于数据中的生命形式。

---

## 展区

### 🌊 深海区
模拟深海环境。黑暗、安静、发光生物。

| 生物 | 说明 |
|------|------|
| 灯笼鱼 | 自带光源 |
| 数据水母 | 半透明，会变色 |
| 深渊章鱼 | 很大，不常出现 |

### 🐠 珊瑚区
明亮、色彩丰富。

| 生物 | 说明 |
|------|------|
| 热带鱼群 | 各种颜色 |
| 珊瑚 | 会缓慢生长 |
| 海龟 | 每天固定时间喂食 |

### 🦈 大洋区
开阔水域。大型生物。

| 生物 | 说明 |
|------|------|
| 鲨鱼 | 三只，有名字 |
| 鳐鱼 | 会飞一样游泳 |
| 金枪鱼群 | 速度很快 |

### 🔮 数字生物区
只存在于数字空间的物种。

| 生物 | 说明 |
|------|------|
| 代码鱼 | 身体是流动的代码 |
| 像素虾 | 8bit风格 |
| 故障海葵 | 会glitch |

## 体验

### 喂食表演
- 珊瑚区：11:00, 15:00

可以摸：海星、海参、小鱼
位置：一楼入口旁

### 海底隧道
走在水下，鱼在头顶游。
位置：大洋区
