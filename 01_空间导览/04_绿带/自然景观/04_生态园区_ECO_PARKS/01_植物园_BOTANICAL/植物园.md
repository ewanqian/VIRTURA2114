## 简介

VIRTURA植物园收藏了2000多种数字植物。

有些是真实植物的数字复制品，有些是只存在于这里的新物种。

安静，适合散步。

---

## 展区

### 🌸 花园区
季节性花卉。每季更换主题。

当前主题：**早春**
- 樱花
- 郁金香
- 水仙

### 🌳 树木区
高大乔木。有些很老了。

| 树种 | 说明 |
|------|------|
| 数据橡树 | 最老的一棵，建站时种的 |
| 代码松 | 四季常青 |
| 记忆柳 | 树叶会播放声音片段 |

### 🌵 沙漠区
仙人掌、多肉。

温度模拟：35°C（体感）
建议：带水

### 🌴 热带区
雨林植物。潮湿、茂密。

| 植物 | 说明 |
|------|------|
| 巨型蕨类 | 比人高 |
| 食虫植物 | 会动 |
| 兰花 | 各种颜色 |

### 🔮 数字植物区
只存在于数字空间的物种。

| 植物 | 说明 |
|------|------|
| 像素花 | 8bit风格 |
| 故障草 | 会glitch |
| 代码藤 | 攀爬，会生成新代码 |
