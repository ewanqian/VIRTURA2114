## 历史遗迹

### 崩溃纪念碑
2050年4月1日，系统崩溃。

纪念碑记录了那一天：
- 失去的数据量
- 幸存者名单
- 重建时间线

位置：静水公园东侧  
门票：免费

### 旧核心遗址
早期服务器的残骸。

现在是露天展览，可以看到：
- 第一代传输设备
- 崩溃时的错误日志（部分）
- 修复痕迹

位置：静水公园北侧  
门票：免费

### 黄金时代街区
保留了2049年的街景。

那时候VIRTURA还在巅峰。
现在这条街是步行街，有咖啡馆和纪念品店。

位置：博物馆群旁边  
门票：免费
