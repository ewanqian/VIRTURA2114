## 区域

### [商业中心](商业中心/)
购物和餐饮。

| 区域 | 内容 |
|------|------|
| 零售区 | 数字生活馆、服装、感官店 |
| 商务区 | 写字楼、会议室 |
| 美食街 | 各种餐厅、咖啡 |

### [文化街区](文化街区/)
艺术和娱乐。

| 区域 | 内容 |
|------|------|
| 唱片公司 | 厂牌、音乐人 |
| Livehouse | 演出场地 |
| 时尚街区 | 服装店、设计师 |
| 画廊 | 虚空画廊、像素空间 |
| 电影院 | 虚空影院 |
| 书店 | 星云出版社、碎片书店 |

## 推荐

### 逛街路线
零售区 → 美食街（午餐）→ 文化街区 → 咖啡

### 文艺路线
画廊 → 书店 → 电影 → Livehouse（晚上）
