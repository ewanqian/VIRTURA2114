## // VIRTUAL_AIRLINE: THE ORIGIN STORY //
### 虚拟航线：起源故事 (2021-2023)

> **[SYSTEM_NOTE]**: 本文档记录了 VIRTURA 项目的真正起源——"虚拟航线"（Virtual Airline）。这是一切的开始。

---

### // THE CONCEPT //

**[BIRTH_DATE]**: 2021.03  
**[CREATOR]**: Code Talker (钱誉文)  
**[ORIGINAL_VISION]**: 通过沉浸式体验讲述计算机技术与数字世界的演变

**[CORE_IDEA]**: 
> "计算机的历史不应该只是枯燥的年表和技术参数。它应该是一段可以'体验'的旅程——从最早的电子信号，到今天的全感知数字世界。"

---

### // THE FOUR WORLDS //

Virtual Airline 的核心是四个连续的"世界"，代表计算机技术的四个阶段：

#### 1. 信号世界 (Signal World)
**[TIME_PERIOD]**: 前网络时代  
**[VISUAL]**: 黑白、像素化、抽象  
**[CONCEPT]**: 计算机技术的黎明，电子信号的孤独闪烁  
**[EMOTION]**: 孤独、原始、纯粹

**[DESIGN_NOTES]**:
*   灵感来自早期的 CRT 显示器和示波器
*   音效：单调的蜂鸣声和电子脉冲
*   互动：用户可以"触摸"信号，感受数据的流动

---

#### 2. 模因世界 (Meme World)
**[TIME_PERIOD]**: 2000s - 2010s（互联网爆发期）  
**[VISUAL]**: 彩色、混乱、信息过载  
**[CONCEPT]**: 网络连接一切，信息如洪水般涌来，迷因（Meme）成为文化基因  
**[EMOTION]**: 兴奋、混乱、迷失

**[DESIGN_NOTES]**:
*   灵感来自早期互联网文化（论坛、博客、社交媒体）
*   音效：嘈杂的人声、通知音、键盘敲击声
*   互动：用户被信息流"淹没"，需要学会"过滤"

---

#### 3. 未知世界 (Potential World)
**[TIME_PERIOD]**: 当下（2021-2023）  
**[VISUAL]**: 半透明、量子叠加态、未定义  
**[CONCEPT]**: 当下的世界，处于一切未知的潜力之中  
**[EMOTION]**: 焦虑、期待、可能性

**[DESIGN_NOTES]**:
*   灵感来自量子力学和薛定谔的猫
*   音效：环境音、白噪音、偶尔的"量子坍缩"声
*   互动：用户的选择会"坍缩"可能性，创造不同的未来

---

#### 4. 未来世界 (Future World)
**[TIME_PERIOD]**: 理想的信息社会（2030s+）  
**[VISUAL]**: 高度秩序化、美学化、乌托邦  
**[CONCEPT]**: 理想的数字乌托邦，早期的太空港雏形  
**[EMOTION]**: 希望、完美、脆弱

**[DESIGN_NOTES]**:
*   灵感来自《银翼杀手》和《攻壳机动队》
*   音效：合成器音乐、空灵的人声
*   互动：用户可以"设计"自己的未来空间

### // TECHNICAL_CHALLENGES //

**[CHALLENGE_001]**: 手工建模效率低下  
*   **问题**: 使用 VRChat 和 Unity，每个世界需要数周甚至数月的建模时间
*   **影响**: 项目进度缓慢，难以实现复杂的交互

**[CHALLENGE_002]**: 沉浸感不足  
*   **问题**: 纯视觉体验缺乏深度，用户很快失去兴趣
*   **影响**: 需要更多的听觉、触觉和叙事元素

**[CHALLENGE_003]**: 单人开发瓶颈  
*   **问题**: Code Talker 一个人无法完成所有工作
*   **影响**: 项目规模受限
