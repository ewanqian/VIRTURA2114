## 航站楼布局

### 到达流程
```
对接口 → 廊桥 → 到达大厅 → 海关检疫 → 行李提取 → 迎宾大厅 → 交通枢纽
```

### 出发流程
```
值机大厅 → 安检 → 候机区 → 登机口 → 廊桥 → 对接口
```

### 到达区
