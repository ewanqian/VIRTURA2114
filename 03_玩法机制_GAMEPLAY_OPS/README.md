## 概述 Overview

VIRTURA 不仅仅是一个可供游览的静态模型，它是一个基于**“搜集-对抗-撤离” (Search, Fight, Evacuate)** 循环的生存探索系统。

我们正在构建一种结合了**AI 文本模拟 (Text Sim)** 与 **沉浸式探索 (Immersive Sim)** 的独特体验。

---

## 🎮 核心协议 Core Protocols

### [PROTOCOL: DEEP DIVE (深潜撤离机制)](PROTOCOL_DEEP_DIVE/README.md)
*   **类型**: 撤离类 (Extraction Shooter / Looter) 变体
*   **概念**: 玩家潜入不稳定的“幻影空间”，搜集珍贵的数据碎片（Loots），并在空间崩塌前撤离。
*   **状态**: 🚧 **概念设计阶段** (Concept Phase)
    *   *灵感来源*: 《逃离塔科夫》、《马拉松》、《潜水员戴夫》
    *   *当前进度*: 我们正在设计基础的循环逻辑与风险收益模型。

### [PROTOCOL: PHANTOM (幻影筛选计划)](PROTOCOL_PHANTOM/README.md)
*   **类型**: AI 生成 / 玩家治理
*   **概念**: 系统利用 AI 生成临时的“幻影空间”，玩家作为“质检员”进入体验，并投票决定该空间是保留还是销毁。
*   **状态**: 📝 **文档策划阶段** (Design Doc)
    *   这是未来的核心 UGC (用户生成内容) 引擎。

### [PROTOCOL: TXT_SIM (文本模拟框架)](PROTOCOL_TXT_SIM/README.md)
*   **类型**: 大语言模型 (LLM) 交互
*   **概念**: 利用 LLM 模拟无限可能的 NPC 对话与环境描述。这是目前成本最低、自由度最高的“最小可行性产品” (MVP)。
*   **状态**: ✅ **原型测试通过** (Prototype Passed)
    *   Ewan 已经进行了三次成功的文本跑团实验（咖啡馆打工、破坏太空站、特工调查）。

## 🎯 愿景 Vision

我们的目标并非复刻一个 3A 级的射击游戏，而是创造一个**“基于语言与感知的 Roguelike 体验”**。

你可能不需要动用扳机，而是动用**语言**、**逻辑**与**情感**来完成撤离。这是 VIRTURA 对“搜打撤”玩法的独特诠释。
