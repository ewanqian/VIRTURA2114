## 概述 Overview

VIRTURA 的核心体验建立在一套被称为 **"S.F.E." (Search / Fight / Evacuate)** 的循环之上。

但这不仅仅是一个游戏机制，它是2050年幸存者们在这个破碎世界中生存的唯一方式。

---

## 🌌 核心协议 Core Protocols

### [02_幻影引擎_PHANTOM_ENGINE](02_幻影引擎_PHANTOM_ENGINE/README.md)
**"世界是生成的，但危险是真实的。"**

*   **定义**: VIRTURA 轨道站的核心超级 AGI。
*   **功能**: 它能根据海量的文化数据（建筑、音乐、艺术、心理学）生成无数个临时的、独立的**“数字空间幻影”**。
*   **特性**: 这些空间像梦境一样脆弱、不稳定，充满了不可预知的美丽和危险。它们是 AGI 对人类集体潜意识的具象化。

### [01_深潜撤离_DEEP_DIVE](01_深潜撤离_DEEP_DIVE/README.md)
**"潜入深渊，带回真相。"**

*   **身份**: 你是**“质检小组” (Quality Inspection Team)** 的一员。
*   **任务**:
    1.  **潜入 (Dive)**: 进入幻影引擎生成的不稳定空间。
    2.  **调查 (Investigate)**: 评估该空间的物理逻辑、情感影响和潜在风险。
    3.  **回收 (Retrieve)**: 搜集珍贵的“信号”碎片（旧时代数据、艺术品、记忆）。
    4.  **撤离 (Evacuate)**: 在空间崩塌前安全返回现实。
*   **起源**: 该机制源自 2025.7.19 的 **DROPFLOW² 演出**，那是人类历史上第一次有组织的深潜行动。

### [03_文本模拟_TXT_SIM](03_文本模拟_TXT_SIM/README.md)
**"语言是唯一的武器。"**

*   **形式**: 基于大语言模型 (LLM) 的无限文本冒险。
*   **描述**: 在这里，没有固定的脚本。你可以用自然语言与任何 NPC 辩论，欺骗虚用部的审查AI，或者用一首诗打动模组临床派的治疗师。

## 🎯 质检小组的使命

你们的筛选并非为了娱乐，而是为了人类文明的存续：

*   **精神疗愈**: 筛选出能缓解焦虑的稳定环境，作为数字处方。
*   **创意工坊**: 寻找能激发创造力的空间，作为灵感源泉。
*   **风险控制**: 销毁那些可能导致意识滞留或精神错乱的“恶性幻影”。

*每一次深潜，都是对未来的投票。*
