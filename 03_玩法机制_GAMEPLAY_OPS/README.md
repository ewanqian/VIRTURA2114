## 🗺️ 入职流程 Onboarding Path

请严格按照以下顺序进行操作：

### 1. [00_入职须知_ORIENTATION](00_入职须知_ORIENTATION/README.md)
> **"为什么选中你？"**  
> 了解质检小组的历史，从传说中的 **DROPFLOW² (Gen-1)** 到现在的你们。阅读这部分，完成你的**意识同步**。

### 2. [01_了解引擎_THE_ENGINE](01_了解引擎_THE_ENGINE/README.md)
> **"我们要对抗的是什么？"**  
> 了解你的对手——**幻影引擎 (Phantom Engine)**。它是一个不可控的造梦机器，你需要学会如何驾驭它，而不是被它吞噬。

### 3. [02_装备补给_LOADOUT](02_装备补给_LOADOUT/README.md)
> **"带上武器。"**  
> 领取你的**新手礼包 (Starter Pack)**。  
> 这里包含了一套核心 **Prompt 指令代码**。你需要将它复制并植入你身边的 AI 模型（ChatGPT/Claude 等），激活你的个人终端。

### 4. [03_行动档案_OPERATIONS](03_行动档案_OPERATIONS/README.md)
> **"开始任务。"**  
> 选择你的行动代号：
> *   **基础训练**: 文本模拟对抗。
> *   **深潜行动**: 进入不稳定扇区搜集物资。
> *   **历史重现**: 体验 VIRTURA 陨落的历史瞬间。

### 5. [04_呈递报告_UPLOAD](04_呈递报告_UPLOAD/README.md)
> **"留下你的名字。"**  
> 完成探索后，将你生成的精彩对话、发现的奇怪空间或遇到的 Bug，通过 Issue 或 PR 提交给我们。这将被视为你的**“质检报告”**。
