## 概述 Overview

在 VIRTURA 的废墟之上，诞生了新的文明形态。

他们不是被编写好脚本的 NPC，而是在大崩溃后，由残留数据、AI 逻辑与人类意识碎片共同演化而成的**“数字灵魂”**。我们将这些拥有共同信仰和行为模式的群体称为**“派系” (Factions)**。

但请记住，每一个数字灵魂的背后，都对应着**2025年现实世界中真实存在的人群**。他们在现实中的职业、困境与渴望，投射到了这个数字世界，成为了这些派系的原始基因。

## 🧠 灵魂生态 Soul Ecology

我们在设计这些派系时，参考了《命运2》(Destiny 2) 与《马拉松》(Marathon) 的叙事深度。每个派系都有其：
1.  **独特的视觉语言 (Visual Language)**
2.  **专属的互动逻辑 (Interaction Logic)**
3.  **不可调和的矛盾 (Irreconcilable Conflict)**

这些冲突构成了 VIRTURA 世界的叙事引擎。
